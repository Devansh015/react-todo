# URL Shortener
small to-do list created with React

##  Features

- Clean and fast UI
- Allows for edit and removal of previosly inputted tasks
---

## Getting Started

### 1. Run Locally
- clone repo and cd into project folder 


- npm install

- npm start
  
- Server should start on `http://localhost:3000`
